Javascript RTree

A fast and memory-efficient RTree implementation for Javascript. It can be loaded with any AMD-loader (e.g. requirejs) , in the browser or in node.

Features

  1. It is fast: search and insertion are very fast. It performs up to 50000-100000 insertions per second on modern browsers.
  2. It is memory-efficient: since RTree's are often used as a low level component in visualization software, you do not want it to spam memory for the garbage collector to clean up. This implementation has a very conservative memory usage profile.
  3. Simple API. Add, insert and remove objects using simple number values.
